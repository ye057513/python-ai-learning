System Prompt vs User Prompt 区别和各自用途
    system prompt:用于设定AI的角色
    user prompt:用于发送用户对于问题的提问与设定
Temperature 参数 的作用
    temperature:控制 AI 回答的随机性，范围 0~2，
    Temperature = 考试的"做题风格"—
    0 = 只选标准答案，每次答卷一模一样
    1.5 = 自由发挥，同一道题三次答出三个版本
Token 是什么，为什么按token计费
    Token:用于表示AI模型的输入输出,也就是AI生成和理解字符所消耗的资源，一个token对应一个字符
